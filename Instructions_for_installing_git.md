### Установка `Git` в операционной системе `Linux`

Шаг 1: Обновление списка пакетов

```bash
sudo apt-get update
```

Шаг 2: Установка `Git`

```bash
sudo apt-get install git
```

Этот процесс скачает и установит необходимые пакеты вместе с зависимостями.

Шаг 3: Проверка версии Git

```bash
git --version
```
